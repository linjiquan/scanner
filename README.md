# 二维码扫描器小程序

一个基于微信小程序的二维码扫描器，支持扫描二维码并将结果发送到后台服务器。

## 功能特点

1. 界面上有一个文本显示框和一个扫描按钮
2. 点击扫描按钮后，调用摄像头扫描二维码
3. 将扫描到的二维码内容显示在文本框中
4. 自动将扫描内容通过HTTP POST方法发送到后台
5. 在文本框中显示发送返回的结果
6. 支持连续扫描，每扫描一次就在显示框中增加一条记录
7. 扫描历史会保存到本地存储中

## 使用说明

1. 使用微信开发者工具导入此项目
2. 在 `index.js` 文件中修改后台接口地址 `https://your-backend-api.com/api/scan` 为实际的后台接口地址
3. 编译并运行小程序
4. 点击"扫描二维码"按钮开始扫描
5. 扫描成功后，二维码内容将显示在文本框中，并自动发送到后台
6. 扫描结果会保存在历史记录中

## 注意事项

1. 请确保您的小程序有相机权限
2. 后台接口需要支持接收POST请求，请求参数为 `{"content": "二维码内容"}`
3. 后台返回的数据格式建议为 `{"result": "处理结果"}`
4. 扫描历史会保存在本地存储中，可以随时查看

## 开发环境

- 微信开发者工具
- 微信小程序基础库 2.14.1 及以上
